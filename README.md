## Building NEM Library Documentation

Install mkdocs

```sh
pip install mkdocs
```

Run mkdocs on server mode

```sh
mkdocs serve
```

Publish NEM Library Documentation to GitHub Pages

```sh
mkdocs gh-deploy
```

More information of mkdocs can be found on the [official website](http://mkdocs.org/).
